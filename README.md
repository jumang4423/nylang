# nylang

![](_img/src.png)

a cute programming language inspired by monkey language in rust

## install & uninstall

```
chmod +x scripts/install.sh && ./scripts/install.sh
```

```
chmod +x scripts/uninstall.sh && ./scripts/uninstall.sh
```

## excution

- show help

```
nylang
```

- run program

```
nylang run <filename>.nyl
```
